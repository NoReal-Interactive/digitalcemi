# digitalcemi
This interactive web application provides users to explore the three-dimensional model of **Cotton Cemí** and gain insight into its historical context through fully immersive and engaging experience.

Developed as a web-based frontend, the application is designed for deployment either through **Electron** (e.g., for kiosk installations) or via a standard web browser.
